# Loaders

Custom asset loaders with optimization and streaming support.

## Files

- **KTX2Loader.ts**: Basis-compressed texture loader
- **ZoneLoader.ts**: Zone-specific asset loader
- **GLTFLoader.ts**: Optimized GLTF model loader

## Features

- Progress tracking
- Cancellation support
- LOD level selection
- Dynamic imports for three.js loaders
