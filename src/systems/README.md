# Systems

Feature systems for collision, navigation, streaming, LOD, culling, instancing, performance, and profiling.

## Subdirectories

- **collision/**: Collision detection and player physics
- **navigation/**: Player movement and zone transitions
- **streaming/**: Zone-based asset streaming
- **lod/**: Level-of-detail management
- **culling/**: Visibility culling (frustum, portal, occlusion)
- **instancing/**: Geometry instancing and batching
- **performance/**: Performance monitoring and adaptive quality
- **profiling/**: Telemetry and profiling tools

## Integration

Systems are coordinated through the main Canvas component and ZoneManager.
