# React Components

UI components for the application.

## Files

- **Canvas.tsx**: Three.js canvas wrapper and main game loop
- **Menu.tsx**: Website menu with social links and contact
- **LoadingScreen.tsx**: Loading progress display
- **PerformanceUI.tsx**: FPS and stats display (dev mode)
- **MobileControls.tsx**: Mobile-only touch controls (not yet implemented as component)

## Usage

Components are used in App.tsx to build the application UI.
