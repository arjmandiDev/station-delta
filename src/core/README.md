# Core Systems

Core Three.js integration systems for scene management, rendering, and camera control.

## Files

- **SceneManager.ts**: Manages Three.js scene, lighting, and environment setup
- **RendererManager.ts**: WebGL renderer with adaptive quality and pixel ratio management
- **CameraController.ts**: First-person camera controls with rotation and positioning

## Usage

These systems are initialized in the Canvas component and provide the foundation for the 3D experience.
